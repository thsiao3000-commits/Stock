# 💼 我的投資組合 + AI 分析

**版本：v1.29.1**（版本號見 `index.html` 的 `APP_VERSION`，並對應 git tag；變更紀錄見 [CHANGELOG.md](CHANGELOG.md)）

追蹤**台股、美股、海外公司債**的個人投資組合工具：逐筆記錄交易、以**加權平均成本法**計算已實現／未實現損益（含台股費稅、股息、債券配息與匯率換算），並可一鍵產生 **AI 技術分析**（針對你的持倉個人化）與**投資組合總評**。

| 工具 | 用途 | 需求 |
|---|---|---|
| 💼 **index.html**（主工具） | 持倉管理＋損益總覽＋走勢圖＋AI 分析 | 瀏覽器；美股需 Twelve Data Key、AI 需 OpenAI Key |
| 🐍 **twse_realtime.py** | 台股**盤中即時**報價輪詢（終端機） | Python 3，免帳號免金鑰 |

## ✨ 功能特色（主工具）

- **三種資產一起管**：
  - **台股** — **上市**走 TWSE 收盤；**上櫃（OTC）**自動改用 FinMind（如 8358 金居），無痛支援兩種市場
  - **美股** — 報價走 Twelve Data（免費方案即可，批量更新）
  - **海外公司債** — 輸入面額／票息／配息頻率／到期日，自動算**配息行事曆、已收息（扣前手息）、應計利息**；市價從銀行對帳單手動更新
- **逐筆交易紀錄**：買進／賣出／現金股息可補登、**編輯**、刪除；台股手續費（0.1425%）與賣出證交稅（0.3%）自動試算、可修改
- **報價更新**：一鍵「🔄 更新所有報價」（台股＋美股＋匯率）；或在持倉明細展開後「🔄 更新此檔報價」**只更新單一個股**，美股單檔免等每分鐘額度
- **損益總覽**：總市值／持有成本／未實現／已實現＋股息息收／總損益，全部**換算新台幣**加總；各市場小計與占比；USD/TWD 匯率自動抓取（12 小時快取）或手動指定
- **投資組合績效**：**XIRR**（年化內部報酬率，money-weighted，由現金流即時計算）；一鍵建立**近一年淨值曲線**（依交易紀錄重建每日投組淨值）並與**加權指數**對照（債券不納入曲線；外幣以目前匯率換算）
- **走勢圖**：近一年日線 ＋ 我的均價虛線 ＋ 買賣點標記，滑鼠懸停顯示「vs 均價」乖離
- **AI 個股分析**（串流即時顯示）：送出 52 週週線＋近 60 日日線＋程式算好的指標（MA5–120、RSI、MACD、KD、**布林通道、ATR**、**OBV 能量潮、量比**、52 週高低、5／20 日均量，且 RSI／MACD／KD 附 **5／20 日前對照**、均線附**斜率與最近黃金／死亡交叉**、現價附**距 52 週高低與乖離率**、另算**量價密集區**當支撐壓力依據，讓 AI 判讀指標「走向」而非只看水位）＋**配息與殖利率**（台股：現金殖利率、配息頻率、連續配息年數、**除息日／發放日**、近 5 年配息趨勢）＋**籌碼面**（台股：三大法人買賣超，最近日／近 5／20 日；美股：機構／內部人持股、分析師動向經 web search）＋**基本面**（台股：月營收年增趨勢、PER／PBR／殖利率；美股：估值／獲利／成長經 web search）＋**消息面**（台美股皆以 **OpenAI web search 即時搜尋**最新新聞並附來源連結，可於設定關閉）＋**大盤對照**（上市配加權指數、**上櫃配櫃買指數**、美股配 SPY）＋**我的持倉成本與損益**，產出含「續抱／加碼／減碼／停損」情境分析的報告，並附一張**操作計畫表**（Markdown 表格，列出各動作的**具體價位**）；未持有標的則改為進場評估
- **AI 投資組合總評**：整個組合（含債券、占比、匯率）的配置／集中度／再平衡建議；Prompt 內含程式精算的 **XIRR、市場／幣別配置占比、未實現／已實現拆分、各檔持有天數、債券距到期年數**，及（若已建立）**近一年淨值曲線 vs 加權指數**結果
- **複製 Prompt（省 token）**：可把送給 OpenAI 的同一份提問一鍵複製，貼到訂閱制聊天機器人（如 Gemini）執行，省下 OpenAI API token；新聞／美股基本面那段需該機器人能上網查詢，台股結構化數據則已內含在 Prompt 中
- **作業中動畫**：AI 產生報告期間，狀態列顯示轉圈動畫＋經過秒數＋階段文字（**思考中 → 撰寫中**），秒數持續跳動讓你確知 AI 仍在運作、不是當機（reasoning 模型在吐出第一個字前可能靜默數十秒）。
- **投資偏好個人化**：設定區可選填「🎯 投資目標與風險偏好」（如存股／波段、可承受回撤、可再投入資金），寫進兩種 AI Prompt，讓「加碼／減碼／停損」等建議**貼合你的目標**而非通用答案（隨 JSON 備份匯出）。
- **分析視角（多人格）**：可切換**價值存股／成長動能／技術短線／風控防守／供應鏈瓶頸獵人**單一風格，或開啟**三師會診**——價值×成長×技術三位分析師各自表態（立場＋理由＋各自價位、不硬湊共識），主持人點出**共識與分歧**後裁決最終操作計畫；分歧本身就是訊號。另可**自訂視角**：把你欣賞的分析師方法論寫成判斷準則，AI 以該風格分析（明標 AI 模擬、非本人觀點）。
- **上次建議檢討**：同一檔重複分析時，自動把**上次報告的操作計畫與結論**＋程式算好的**期間實際走勢**（漲跌幅、最高最低）餵回 AI，要求先誠實檢討上次判斷的對錯再給新建議——AI 要對自己說過的話負責。
- **觀察清單快篩**：「還沒買、在觀察」的標的（上限 12 檔）一鍵抓一年價量、算指標**並排比較**（趨勢／均線交叉／RSI／MACD 動能／量比／距 52 週高），可讓 **AI 排序評分**找出值得深入的對象，再一鍵帶入完整個股分析；消息面另會概查**社群輿情**（台股 PTT／Threads、美股 X／Reddit，並提醒情緒過熱常為反指標）。
- **KOL 選股雷達（三模式）**：**⚡ Grok 直查 X**（填 xAI Key，`x_search` 鎖定 KOL **本人帳號**近 60 天貼文直讀——一手最即時，約 US$0.02–0.05/次）；**🔍 網路搜查**（OpenAI web search 靠二手轉述，滯後數天～數週）；**✂️ 貼文擷取**（貼上原文本地抽 `$代號`，零費用）。表格皆附**發文日期＋當時價位＋來源連結**，追不追由你對照現價自行判斷；一鍵加入觀察清單接快篩動線。雷達報告顯示在**雷達面板內**（含複製／下載），與 ③ 的快篩總評、個股分析**可同時對照**、互不覆蓋。
- **報告可保存＋歷史切換**：報告開頭自動標記**產生時間、AI 模型、工具版本**；一鍵**複製**或**下載為 `.md`**（檔名帶標的與日期）；自動保留**最近 10 份**於本機（重新整理不消失、開頁還原上次報告），下拉選單可切換對照前後觀點變化。
- **引用來源清單**：開啟 web search 時，報告尾端自動附上**去重後的新聞來源連結**（接住 API 的 url_citation 引用事件，不再依賴模型自願寫出網址）。
- **列印友善**：`Ctrl/Cmd+P` 列印或存 PDF 自動轉**白底黑字**、隱藏設定區與按鈕（**API 金鑰不會被印出**）。
- **資料匯出／匯入**：一鍵匯出全部持倉與設定為 JSON（備份、跨裝置／跨瀏覽器搬移），匯入即還原；可選擇是否包含 API 金鑰。
- 所有資料存在**瀏覽器 localStorage**，不上傳任何伺服器

## 🚀 快速開始

直接用瀏覽器開啟 `index.html`（Mac 可在終端機 `open index.html`），或開已部署的線上版（見下方「部署到 GitHub Pages」）。

1. 展開頂部「**🔑 API 與參數設定**」：
   - **Twelve Data API Key** — 美股報價用。到 [twelvedata.com](https://twelvedata.com) 免費註冊取得（800 credits／天、8／分鐘，個人使用綽綽有餘）。只用台股＋債券可不填。
   - **FinMind Token** — 台股**上櫃**用，**選填**。免填即可運作；若被限流再到 [finmindtrade.com](https://finmindtrade.com) 免費註冊取得 token 提高額度。
   - **OpenAI API Key** — AI 分析用。可勾選「記住」存於本機。
2. 「**② 我的持倉**」→「＋ 新增持倉」：選市場、填首筆買進（債券填面額／票息／到期日等）。
3. 按「**🔄 更新所有報價**」：台股、美股、匯率一次更新。
4. 點任一持倉列可展開：補登**買進／賣出／現金股息**、債券**更新市價**、刪除。
5. 「**③ 走勢圖與 AI 分析**」：選標的 →「📊 抓取一年資料」看走勢 →「🤖 產生個股 AI 分析」；或直接「💼 產生投資組合總評」。

## 📊 損益計算規則

- **加權平均成本法**（與台灣券商 App 一致）：買進攤入均價；賣出依當時均價認列**已實現損益**
- **台股未實現損益**已扣**預估賣出費稅**（手續費＋證交稅，費率可在設定調整）
- **現金股息**手動補登（台股價格未還原除權息，補登後總報酬才完整）
- **債券**：總損益 ＝ 已收息（扣買進時支付的前手息）＋（市價 − 買進成本）；配息日由**到期日回推**產生；另列出尚未配發的**應計利息**供參考；到期自動以 100% 面額計

## 🤖 AI 分析用到的資料

個股分析送給 AI 的資料分四類；**所有數字都由程式精算後才交給 AI，AI 只負責「解讀」不負責「計算」**：

1. **程式精算的指標**：技術面（MA、RSI、MACD、KD、布林通道、ATR、OBV、量比、量能，含 **5／20 日前對照、均線交叉偵測、乖離率與量價密集區**）；台股另含產業別、配息（殖利率、除息／發放日、連續配息年數）、三大法人買賣超、月營收年增、本益比(trailing)／PBR。
2. **原始價量**：近 52 週週線＋近 60 日日線 OHLCV、同期大盤對照（上市配加權指數、上櫃配櫃買指數、美股配 SPY）。
3. **AI 即時 web search**：最新新聞（附來源連結）；美股的估值（trailing／forward 本益比）、營收成長、分析師評等與機構持股。
4. **你的持倉**：加權平均成本與損益（存於本機）。

> 投資組合總評則使用各持倉的損益與占比、總計與匯率（不做新聞搜尋）。

## 🌐 部署到 GitHub Pages

> **先理解兩件事**
> 1. **資料永遠在你自己的瀏覽器**：Pages 只負責把 HTML 送到瀏覽器，之後 API Key、資產明細、交易紀錄全部存在你裝置的 localStorage，API 請求由瀏覽器**直連** OpenAI／TWSE／Twelve Data，**不經過 GitHub**。GitHub 看不到你的任何資料。
> 2. **發佈出來的「網站」是公開的**：知道網址的人都能打開（這支工具是 BYOK，訪客得自己填自己的 key，碰不到你的）。把 repo 設 private 只隱藏原始碼、不會讓網站變私密。

**步驟**

1. **把 repo 改為 Public**（免費方案的 Pages 只能從 public repo 發佈）。
   repo 頁面 → **Settings → General → 最下方 Danger Zone → Change visibility → Public**。
   *（本專案程式碼裡沒有任何祕密：`.env` 已 gitignore、API Key 不寫在程式碼中，故公開原始碼是安全的。若想維持 private，需 GitHub Pro 以上。）*
2. **Settings → Pages → Build and deployment**
   - Source：**Deploy from a branch**
   - Branch：**`main`** ／ 資料夾 **`/(root)`** → **Save**
3. 等約 1 分鐘，網站上線於：
   **`https://thsiao3000-commits.github.io/Stock/`**
   （`index.html` 為預設入口，網址不必帶檔名，乾淨好記。）

**上線後的安全注意**

- **金鑰**：只存在你自己的瀏覽器。注意儲存行為分兩種——**OpenAI Key** 勾「記住」才存；**Twelve Data／FinMind／Finnhub／xAI** 金鑰**輸入即自動儲存**於 localStorage。建議給這工具用的 key 都設**用量上限**；在公用電腦上除了別勾「記住」，**其他金鑰也不要輸入**（或用完手動清空欄位）。
- **共用 origin** ⚠️：同帳號底下所有 Pages 專案共用 `https://thsiao3000-commits.github.io` 這個 origin，彼此的 localStorage 互通。若日後在同帳號掛其他頁面，那些頁面的 JS 讀得到本工具存的 key。化解：改用**自訂網域**取得獨立 origin，或別在同帳號放來路不明的頁面。
- **想連「網站本身」都不公開**：改用 Cloudflare Pages／Netlify（可從 private repo 部署、原始碼不公開，並能加存取密碼）。

## 🐍 台股盤中即時報價 CLI

收盤前想看即時價，用 CLI（TWSE MIS 快照，官方頁面 5 秒更新一次）：

```bash
python3 twse_realtime.py                # 查預設清單（2330 2317 0050）
python3 twse_realtime.py 2330 2317     # 查指定股票（一次）
python3 twse_realtime.py --loop 2330   # 每 5 秒輪詢，Ctrl+C 停止
```

上市自動帶 `tse_` 前綴；**上櫃**請寫成 `otc_5483` 格式（主工具僅支援上市，上櫃可用此 CLI 查價後手動對照）。

## ❓ 常見問題

- **美股報價失敗／很多檔怎麼辦？** 確認已填 Twelve Data Key。免費版每分鐘 8 credits（每支 1），工具會**自動分批（每 7 支一批）、批間隔 60 秒**並顯示倒數，所以建 10 支、20 支美股也能一次按完，只是多等幾十秒；若仍被限流會自動等一分鐘重試一次。
- **台股查無報價？** 上市走 TWSE、上櫃（OTC）自動改用 FinMind，兩者皆支援。若仍查無，多半是代號錯誤、新上市櫃資料不足，或 FinMind 暫時限流（可在設定填免費 FinMind token）。
- **匯率不對？** 在設定的「USD/TWD 匯率」直接填想用的數字（留空＝自動）。
- **AI 分析很慢？** 推理深度選「低」會快很多；報告以串流逐字顯示，不用等全部完成。產生期間狀態列會顯示轉圈動畫＋經過秒數＋「思考中／撰寫中」，只要秒數持續跳動就代表 AI 正在運作、沒有當機（reasoning 模型在輸出第一個字前可能先靜默思考數十秒）。
- **AI 報告寫到一半就停／被截斷？** 多半是達到輸出長度上限（推理深度越高，推理本身越吃額度）。本工具上限已提高，並會在截斷時於狀態列以「⚠️」提示；遇到時把「AI 推理深度」調低再產生一次即可。
- **金鑰欄位被自動填成別的 key？** 已加入防瀏覽器／密碼管理員自動填入的處理（欄位載入時唯讀、點擊才解鎖）。若你的瀏覽器仍硬要填，請到瀏覽器密碼管理員刪掉對 `github.io` 儲存的密碼，或關閉該站自動填入。
- **資料會不會不見？** 持倉存在瀏覽器 localStorage——換瀏覽器／清除網站資料就沒了。請用設定區的「**⬇️ 匯出 JSON**」定期備份；換裝置或清快取後用「**⬆️ 匯入 JSON**」還原。
